### 1. Background
+ 자전거 대여소 자치구별 설치 건수 현황
![Alt text](/img/1stationNum.jpg '자치구별 설치건수')
                         <2021년 1월 31일 기준>

자전거 대여소가 가장 많이 설치된 자치구는 강서구이지만 지하철 주변에 가장 많이 설치되어 있는 자치는 해당 자치구에 설치되어있는 대여소 기준 35%가 역출구에 위치해 있다. 이는 동작구의 특성상 많은 지하철역이 존재할 수있기 때문에 더 분석 필요함.
간단하게 확인해보자!
